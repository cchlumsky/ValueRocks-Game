VALUE ROCKS



Game Instructions

A random number will be automatically produced at the beginning of the game.

The player will need to click one of the four "Value Rocks" below to produce a specific value that will be added to your tally.

A win will be added to your counter only if you are able to match the random score exactly by using the value of each rocks.

Each new game will not only produce a new number but it will also change the value of each rock. GOOD LUCK!!
